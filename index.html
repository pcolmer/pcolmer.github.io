Relative History

Genealogy at your fingertips

release
06 Jul 2016

We’re pleased to announce the release of version 2.4.20 for Windows 8.1, Windows Phone 8.1, Windows 10 and Windows 10 Mobile.

As ever, please check the Help page for details on how to contact us if you need assistance or if you have any suggestions for future features.

Features

  • Written for Windows 8.1 and Windows Phone 8.1
  • Compatible with Windows 10 and Windows 10 Mobile
  • Imports GEDCOM 5.5 and 5.5.1 files
  • Supports international character sets
  • Supports media files
  • Displays maps for addresses and places in events
  • Easy to search across the data
  • OneDrive support includes shared files
  • Free! (*)

(*) In-app purchases may be used at a future time when new functionality, such as editing all data, is made available in the app.
