Welcome

This site aims to be a comprehensive guide to Relative History. We’ll cover topics such as importing a GEDCOM file, troubleshooting import problems and navigating around your data in the app.

Windows 10/Windows 10 Mobile

The current release of Relative History has primarily been designed for use with Windows 8.1 and Windows Phone 8.1. This means that, although the app can be used without problems on Windows 10 and Windows 10 Mobile, there are some user interface differences due to the changes that Microsoft made for Windows 10. Those differences will be highlighted in this documentation.

So what is Relative History?

Relative History imports your genealogy information by using the industry-standard data interchange format, GEDCOM. Once imported, you can easily browse and search it. You can import multiple GEDCOM files and explore the connections between individuals, families and events, viewing associated images or other media files.

If you come across anything along the way that we haven’t covered, or if you know of a tip you think others would find handy, please file an issue.
