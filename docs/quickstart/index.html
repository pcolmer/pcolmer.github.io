Quick-start guide

Relative History can be installed from the Windows Store:

  • Windows 10
  • Windows 8.1
  • Windows Phone 8.1

When the app is first run it installs a sample database containing the genealogical history of the British Royal Family. This is just an example database and can be removed if it isn’t needed or wanted.

You can then import one or more GEDCOM files or Relative History databases.
