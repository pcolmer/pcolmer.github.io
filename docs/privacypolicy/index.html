Privacy Policy

Telemetry data

Depending on the version of the app, Relative History may collect anonymous data that is used to improve features in the app.

This functionality is enabled by default but can be turned off with the Collect telemetry setting. The anonymous data is shared with Microsoft as they provide the telemetry functionality used by Papillon Productions.

If there isn’t a Collect telemetry setting, this feature isn’t present in the app.

Crash data

Other versions of the app may only send crash data. This is also anonymous data and is used to help fix problems with the app. The data is shared with Microsoft/HockeyApp as they provide the crash analysis service used by Papillon Productions.

Some versions of the app will send this crash data automatically when the app starts. If you want to be asked whether or not you want to send the crash data, this can be changed with the Send crash data automatically setting.

Other information

No other information is collected by the app.

The app accesses the Internet for adverts and when directed to by the user for accessing files stored on OneDrive.

This policy is subject to change without notice. Please check this page for the latest privacy policy. If you disagree with this policy, please discontinue use of the application.
