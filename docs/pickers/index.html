Pickers

For some information, e.g. citations, you need to specify the record that you are citing from first. Relative History standardises on how this work through the use of a picker. The initial view of a picker is like this:

From here, you can:

  • Select a recent record
  • Search for a record
  • Create a new record

If you tap on a recent item or a searched item, the screen will update to show some of the information from that record to allow you to confirm that you’ve found the record you were looking for:

Once you’ve picked or created the record, you will be taken back to the screen that then references the record.
