Getting Help

Need help with Relative History? Try these resources.

Crashes

As of version 2.4.15, the app will automatically send anonymous crash data to us the next time that the app is run. This data helps to narrow down the cause of a crash but it may not be enough information for us to reproduce it.

If you experience the app suddenly quitting, please send an email to Papillon Productions Support. We will try to associate your experience with the crash data we receive so that we can then work more closely with you to fix the problem as quickly as possible.

Documentation

Our guide to the features of Relative History and how to get started with it.

UserVoice

If you are having a problem or have feedback about the app, UserVoice is the best way to get in touch.

Support email

If you prefer to use email, please remember to include as much information as possible about the problem you are having. If possible, please consider including the GEDCOM file you are using with the app as that often helps us to troubleshoot the problem faster.

Please note that any GEDCOM files sent to us are kept confidential and deleted after the problem is resolved.

Facebook

The Facebook page for Relative History. You can ask questions here or “like” the page to stay informed on what is happening.

@PapillonSupport on Twitter

The official Papillon Productions Twitter account. It’s not used very often, so try the above first.
