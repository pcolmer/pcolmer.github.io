Relative History 2.5.0 released

release
28 Apr 2016

After what has seemed like quite a long time since the incremental improvements to version 2.4, we’re pleased to announce the release of version 2.5.0 for Windows 8.1, Windows Phone 8.1, Windows 10 and Windows 10 Mobile.

From this release onwards, more details around changes in each version will be documented on this web site. That said, the main change in this release is a new feature, namely the Relationship Calculator.

As ever, please check the Help page for details on how to contact us if you need assistance or if you have any suggestions for future features.
